SeLaLib

Bibliothèque modulaire pour la simulation du modèle gyrocinétique par une méthode semi-Lagrangienne.

Dans le cadre de l'action d'envergure nationale Fusion, l'EPI Calvi développe en collaboration avec le CEA de Cadarache le code GYSELA de simulation gyrocinétique pour la simulation de la turbulence dans les plasmas de Tokamaks. Le développement et le test de nouvelles méthodes numériques se fait en général sur différents modèles simplifiés avant son intégration dans GYSELA. Aucune spécification n'est mise en oeuvre dans cet objectif, ce qui implique plusieurs réécritures du code avant l'intégration définitive, ce qui est un processus lourd et inefficace. SeLaLib est une API pour les composants numériques de base mise en oeuvre également dans GYSELA et un canevas parallèle unique permettant de rationaliser, de fiabiliser et de simplifier le travail de développement.

Modular library for the gyrokinetic simulation model by a semi-Lagrangian method.

As part of french action Fusion, Calvi INRIA projetct developed in collaboration with CEA Cadarache GYSELA simulation code for gyrokinetic simulation of plasma turbulence in Tokamaks. Development and testing of new numerical methods is generally done on different simplified models before its integration into GYSELA. No specification is implemented for this aim, which involves several rewriting code lines before the final integration, which is cumbersome and inefficient. SeLaLib is an API for components of basic numerical implementation also in GYSELA, a framework for parallel single streamline built in order to improve reliability and simplify the work of development.
