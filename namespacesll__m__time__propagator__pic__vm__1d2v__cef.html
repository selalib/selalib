sll_m_time_propagator_pic_vm_1d2v_cef Module Reference

Description

Particle pusher based on Hamiltonian splitting proposed by Crouseilles, Einkemmer, Faou for 1d2v Vlasov-Maxwell.

Author
Katharina Kormann, IPP

MPI parallelization by domain cloning. Periodic boundaries. Spline DoFs numerated by the point the spline starts. Reference: Kraus, Kormann, Sonnendr√ºcker, Morrison: GEMPIC: Geometric ElectroMagnetic Particle-In-Cell Methods
