sll_m_compression Module Reference

Description

Module providing an F90 interface to the ZFP compression library.

ZFP http://computation.llnl.gov/projects/floating-point-compression In addition it provides simple threaded (de)compression routines. Important: This module uses C-like 0-based indexing!

Author
Klaus Reuter, Max Planck Computing and Data Facility (MPCDF)
