Detailed Description

This module defines a hexagonal mesh.

Author
  • Laura S. Mendoza (mela@.nosp@m.ipp..nosp@m.mpg.d.nosp@m.e)

  • Charles Prouveur (prouv.nosp@m.eur@.nosp@m.math..nosp@m.univ.nosp@m.-lyon.nosp@m.1.fr)

    An hexagonal mesh is a triangular mesh with an hexagonally-shaped boundary subdivided in equilateral triangles. It has the

particularity of being generated by 3 non-orthogonal vectors.
